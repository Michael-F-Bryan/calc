calc::syntax

Module calc::syntax::visit [−] [src]

Syntax tree traversal.

Each method of the Visitor trait is a hook that can be overridden to customize the behavior when visiting the corresponding type of node. By default, every method recursively visits the substructure of the input by invoking the right visitor method of each of its fields.

Use the walk_*() functions to continue traversing the AST in the default traversal order.
